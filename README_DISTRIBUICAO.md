# PAICS - Sistema de Análise de Imagens Veterinárias

## Instalação

1. Extraia todos os arquivos da pasta em um local de sua preferência
2. Execute `PAICS.exe`
3. Na primeira execução, você será solicitado a configurar sua API Key do Google Gemini

## Obtendo API Key

1. Acesse: https://aistudio.google.com/app/apikey
2. Faça login com sua conta Google
3. Clique em "Create API Key"
4. Copie a chave gerada
5. Cole no campo solicitado pelo PAICS

## Requisitos de Sistema

- Windows 10 ou superior
- Conexão com internet (para acessar a IA)
- 2GB de RAM mínimo
- 500MB de espaço em disco

## Uso

1. Execute `PAICS.exe`
2. O navegador abrirá automaticamente
3. Faça upload do PDF com imagens do exame
4. Aguarde a análise da IA
5. Revise e edite o laudo gerado
6. Baixe em PDF ou Word
